🖌️ Painting

  • Click & drag — paint cells with the active plant
  • Shift + click & drag — erase cells
  • Ctrl + click & drag — straight-line mode; axis locks on first movement
  • Right-click — open context menu

🌱 Selecting a Plant

  • Click any plant in the toolbar to select it
  • Use Quick Plant in the right-click menu for common plants
  • Open the 📋 Spacing Guide to browse all plants and click one to select it

📐 Grid

  • Each cell = 1 sq ft
  • Heavier grid lines mark every 5 ft
  • Change garden size with the Width × Height controls and click Apply

💾 Saving & Exporting

  • Export — saves a .garden file (gzip-compressed)
  • Import — loads a .garden file and restores the grid
  • Export PNG — downloads a rendered image of the garden
  • Export PDF — opens the browser print dialog
